$rootScope.Scope - type in module ng

root Scope는 '$rootScope'로 '$injector'에서 찾을수 있다. Child scope는 '$new' 메소드를 사용하여 생성된다.(대부분의 스코프는 컴파일된 HTML template이 실행되면 자동으로 생성된다.) 심층적인 소개와 사용예제는 'Scope guide'를 참조하면 된다.

Inheritance(상속)

스코프는 부모스코프로부터 상속된다. 그예는 아래와 같다.

Usage(사용법)

$rootScope.Scope([providers], [instantceCache]); Arguments(인자) - Providers(선택기능) / Object / 서비스 Factory 맵이고 현재 범위 내에서 제공됩니다. 기본은 'ng'입니다. - instanceCache(선택기능) / Object / 'providers'로 부터 append, override 서비스가 추가된 사전 인스턴스화 서비스를 제공합니다. Return(반환값) -Object / 새로 생성된 scope Methods - $new(isolate, parent); 새로운 자식 scope를 생성한다. 부모 scope는 $digest() 이벤트로 전파한다. scope는 $destroy()를 사용하여 scope 계층 구조를 제거할 수 있다. $destory()는 대상 scope와 그 하위 scope 스코프를 호출하고, 이 범위에 대한 요구와 그 자식 scope는 부모와 영구적으로 분리할때 scope에서 호출된다. -
